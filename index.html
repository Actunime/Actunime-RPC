Options de démarrage

Configuration du statut

Vous devez vous rendre sur https://discord.com/developers/applications créer une nouvelle application puis copier son Identifiant APPLICATION ID et la coller ci-dessus.

Pour avoir une clé vous devez ajouter une image sous la rubrique "Rich Présence" puis "art assets" de votre application puis vous prenez ensuite le nom de l'image.

Le texte est une mini description quand vous passez la souris sur l'image.

Lien obligatoire sinon le bouton apparait pas sur le statut.

Streaming twitch

Le mode streamer permet de détecter automatiquement si la chaine twitch renseigné est en live et d'appliquer les paramètres ci-dessous, une actualisation sera faite toute les minutes.
